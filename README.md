# Outdoor Planner Pro 🏔️🏂

Aplicación de escritorio moderna diseñada en Python para la planificación segura de rutas de montaña, trekking, trail running y splitboard.

## Características
- Cálculo estimado de tiempos de marcha basado en distancia y desniveles.
- Checklist dinámico de seguridad según la actividad seleccionada.
- Interfaz gráfica moderna en modo oscuro (CustomTkinter).
- Exportación automática del plan de ruta a formato de texto (.txt) para uso offline.

## Tecnologías utilizadas
- Python 3
- CustomTkinter (UI)
- Git & GitHub
